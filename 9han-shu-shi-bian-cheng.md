# 9.函数式编程

## 一、高阶函数

### 1.map函数

##### （1）作用

`map`将传入的函数依次作用到序列的每个元素，并把结果作为新的`Iterator`返回。

##### （2）使用方法

`map()`函数接收两个参数，一个是函数，一个是`Iterable`，即map（函数名，`Iterable`）

##### （3）举例

```py
>>> def f(x):
...     return x * x
...
>>> r = map(f, [1, 2, 3, 4, 5, 6, 7, 8, 9])
>>> list(r)
[1, 4, 9, 16, 25, 36, 49, 64, 81]
```
